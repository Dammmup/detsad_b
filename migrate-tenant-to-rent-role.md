# Миграция: Преобразование поля tenant в роль rent

Данный документ описывает процесс миграции, который преобразует поле `tenant` у пользователей в новую роль `rent`.

## Что делает миграция

1. Находит всех пользователей, у которых поле `tenant` равно `true`
2. Изменяет их роль с текущей на `rent`
3. Удаляет поле `tenant` из всех пользователей

## Как запустить миграцию

```bash
cd backend
npm run migrate-tenant-to-rent
```

## После миграции

1. Пользователи с ролью `rent` будут отображаться на вкладке "Арендаторы" на странице "Сотрудники"
2. Пользователи с ролью `rent` не будут отображаться в основных списках сотрудников
3. При добавлении новых арендаторов автоматически будет устанавливаться роль `rent`
4. В списке ролей для обычных сотрудников роль `rent` больше не будет отображаться

## Обратная совместимость

Поле `tenant` полностью удалено из модели пользователя, так как теперь его функцию выполняет роль `rent`.
